
AdventureWorks Försäljningsanalys

Beskrivning:
Detta projekt analyserar AdventureWorks2019-databasen med Python och Jupyter Notebook. Analysen innehåller 7 visualiseringar som besvarar affärsfrågor om produkter, försäljning, kunder och regioner.

Visualiseringar:
-Antal produkter per kategori – visar vilka kategorier som har flest/minst produkter.

-Försäljning per produktkategori – vilka produktkategorier genererar mest intäkter.

-Försäljningstrend över tid – hur försäljningen utvecklas månad för månad.

-Försäljning och antal ordrar per år – jämför åren, både försäljning och antal ordrar.

-Top 10 produkter efter försäljning – vilka produkter som säljer mest och vilken kategori som dominerar.

-Försäljning och antal kunder per region – vilka regioner är starkast/svagast.

-Genomsnittligt ordervärde per region och kundtyp – skillnader mellan individuella kunder och företag.

Installation:

Klona repo:
git clone -REPO URL-

Gå till mappen:
cd AdventureWorks-analys

Installera beroenden:
pip install -r requirements.txt

Körning:
Öppna AdventureWorks_Analysis.ipynb i Jupyter Notebook och kör cellerna för att se analysen och visualiseringarna.
